# Different compression type
1. x - extract
2. f - read from a file
3. v - verbose output (list all files)

*  *.tar file

`` tar xf foo.tar ``

*  *.gz file

`` tar xzf foo.gz ``

*   *.xz file

`` tar xz foo.xz ``


*  *.bz2 file

`` tar xjf foo.bz2 ``
